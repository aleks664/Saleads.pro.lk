Создание бота

1 Создаете сообщество
2 Заходите в Управление сообществом - Работа с API и нажимаете “Создать ключ”
3 Проставляете галочки на первые 3 пункта и получаете Api токен. Затем обязательно, в разделе "Сообщения - Настройки для бота" включите возможности бота
4 Указываете ID сообщества (только цифры), API токен и сохраняете. Ваш бот готов
Название используется исключительно для отображения в личном кабинете
API ключ полученный при создании бота
Только цифры идентификатора группы

Бот может раз в сутки присылать 3 оффера пользователям
