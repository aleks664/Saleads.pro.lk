Авторизация

Все запросы API должны содержать API-токен

Все запросы API должны содержать HTTP-заголовок Accept: application/json

Все успешные запросы API возвращают HTTP-статус 200

Передача API-токена возможна двумя способами:

1. Предача API-токена в GET парамете token, например:

GET /api/v1/offer?offset=0&limit=1000&token=2ltgXQM64o4pa66W5tkVy8EcGUiEwlcE8oJJdA80fnYhPhpGiYLXPVRPOiaRN7Jj HTTP/1.1 Host: my.saleads.pro Accept: application/json

2. Предача API-токена HTTP-заголовке Authorization, например:

GET /api/v1/offer?offset=0&limit=1000 HTTP/1.1 Host: my.saleads.pro Accept: application/json Authorization: Bearer 2ltgXQM64o4pa66W5tkVy8EcGUiEwlcE8oJJdA80fnYhPhpGiYLXPVRPOiaRN7Jj

В случае отсутствия или неверно указанного токена будет получен ответ с сообщением об ошибке. Код ответа 401

{"error": "Unauthenticated."}
